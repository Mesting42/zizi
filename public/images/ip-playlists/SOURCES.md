# Sanrio 主题歌单封面来源

Hello Kitty 与库洛米主题各使用 16 张不同封面。素材来自 Sanrio 官方角色页、
官方数字内容页与角色官方社交内容，下载到项目本地以避免外链失效。

- Hello Kitty 官方角色页：https://www.sanrio.co.jp/characters/hellokitty/
- 库洛米官方角色页：https://www.sanrio.co.jp/characters/kuromi/
- Sanrio 官方数字内容：https://www.sanrio.co.jp/digital/
- Sanrio 官方线上背景：https://www.sanrio.co.jp/digital/digital-web-onlinewallpaper/

本轮已移除不适合作为歌单封面的购物横幅、联名商品图和大段文字海报，
替换为角色设计画廊、家人朋友角色图和官方社交插画。
